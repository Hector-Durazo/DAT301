Final Project¶

Due 11:59pm, Tuesday, December 8, 2020¶

You are required to work in groups of 2~3 members (you can continue with your teams in the midterm project). Please include the names and student ID (ASU network ID) of all teambers. Only team leader needs to submit required documents.

The task is similar to that of Midterm Project: work on project of your choice, but this time you need to do it in Python. It is okay to take your topic in Project 1 and do programming in Python, as long as you also upgrade it (for example, answer more questions than you did in the midterm). Furthermore, you can even do some of the previous labs, as long as you convert them into Python, and broaden or deepen the analysis. I do not expect as much of complicated coding as in Midterm Project, since we did not cover Python as much as we did R.

You should include some plots (for example, using matplotlib library) as well as some data manipulation with pandas.

You can create Python script files (.py), but you should also have a report in Jupyter notebook, i.e. ipynb file.

Apart from ipynb file, you also need to upload html and pdf versions of it. You can do this in two ways.

One way, when you have your ipynb open in Jupyter, go to File $\rightarrow$ Download as $\rightarrow$ HTML (.html). Once you create html file, your internet browser should be able to convert html to pdf (at least Google Chrome does that).

Alternativaly, to convert ipynb to html and pdf, open Anaconda prompt. Then, in the prompt, if converting into html, type

jupyter nbconvert --to html file_name_with_full_path

For example, if I want to convert my file final.ipynb into html, which is located in the folder /Users/slan/Teaching/ASU/DAT301/Projects/Final.ipynb, I would type

jupyter nbconvert --to html /Users/slan/Teaching/ASU/DAT301/Projects/Final.ipynb

NOTE: If a folder in the path has name that includes blank space, you would either need to rename the folder or else: first change current working directory where the file is, and then use the above command, but only with the file name (including extension). For example, if I am currently in the folder

(base) slan@XXX ~ %

and my file Final.ipynb is in the folder /Users/slan/Teaching/ASU/DAT301/Projects (note the spaces in the names of the last two folders), then, I would change directory to the last folder by typing

cd ~/Teaching/ASU/DAT301

Then, to convert my Jupyter notebook Final.ipynb (which is now in the current directory) into html, I type

jupyter nbconvert --to html Final.ipynb

To convert ipynb file into pdf, you do the same as above, but replace word "html" with "pdf".
