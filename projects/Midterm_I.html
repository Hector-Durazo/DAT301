Example

For example, consider a data set with human activity from UC Irvine machine learning repository. Use hierarchical clustering and/or k-means clustering to come up with a way of distinguishing between various human activities collected in the sample.

The data set can be found at http://archive.ics.uci.edu/ml/datasets/Smartphone-Based+Recognition+of+Human+Activities+and+Postural+Transitions.

The link with the video on how the data were collected can be found here: https://www.youtube.com/watch?v=XOEN9W05_4A

Consider the clustering problem.

set.seed(1234)
x <- rnorm(12,mean=rep(1:3,each=4),sd=0.2)
y <- rnorm(12,mean=rep(c(1,2,1),each=4),sd=0.2)
plot(x,y, xlim=c(0.5,3), ylim=c(0.5,2.6))
text(x+0.1, y+0.05, label=seq(1:12)) ## including labels (indices)

Now we create a data frame with x and y and make cluster with hclust function.

 dataFrame <- data.frame(x=x,y=y)
distxy <- dist(dataFrame)
hClustering <- hclust(distxy)
plot(hClustering)

It would be convenient to distinguish certain clusters by colors. For that purpose we can utilize a modified version of hclust function, called myplclust, which colors the numbers according to clustering.

myplclust <- function(hclust, lab = hclust$labels,
lab.col = rep(1, length(hclust$labels)), hang = 0.1, ...) {
## modifiction of plclust for plotting hclust objects *in colour*! Copyright 
## Eva KF Chan 2009 Arguments: hclust: hclust object lab: a character vector 
## of labels of the leaves of the tree lab.col: colour for the labels;
## NA=default device foreground colour hang: as in hclust & plclust Side
## effect: A display of hierarchical cluster with coloured leaf labels.
y <- rep(hclust$height, 2)
x <- as.numeric(hclust$merge)
y <- y[which(x < 0)]
x <- x[which(x < 0)]
x <- abs(x)
y <- y[order(x)]
x <- x[order(x)]
plot(hclust, labels = FALSE, hang = hang, ...)
text(x = x, y = y[hclust$order] - (max(hclust$height) * hang), 
     labels = lab[hclust$order], col = lab.col[hclust$order], srt = 90, 
     adj = c(1, 0.5), xpd = NA, ...)
}

Now we can re-plot the clustering dendrogram with myplclust.

myplclust(hClustering, lab = hClustering$order, 
          lab.col = rep(c("red","green","blue"), each = 4))
